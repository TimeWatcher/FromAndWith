# FromAndWith
This is a MVVM android project of social app written in Kotlin with Jetpack Compose.

### Technology Stack
* Language: Kotlin
* UI: Jetpack Compose + Accompanist
* DI: Hilt
* Network: Retrofit2
* Asyn: RxJava2
* Image Loader: Coil

### UI Design
The app follows Material You design.
